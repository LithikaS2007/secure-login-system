# Secure Login System

A simple secure login system built using Flask.

## Features
- User registration and login
- Password hashing using bcrypt
- Protection against SQL injection
- Session management
- Login attempt limiting
- Input validation

## How to Run

1. Install dependencies:
   pip install flask bcrypt

2. Run:
   python app.py

3. Open browser:
   http://127.0.0.1:5000
